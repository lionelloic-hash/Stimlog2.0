# Stimlog 2.0 - Python Version

Koffein-Tracker mit Minigames in Python.

## Installation

```bash
pip install -r requirements.txt
python app.py
```

Öffne dann `http://localhost:5000` im Browser.

## Spielstart (Desktop mit Pygame)

Bearbeite `games.py` um eine der Game-Klassen zu instanziieren und auszuführen.

## Funktionen

- ☕ Koffein-Tracking (Kaffee, Tee, Cola, Energy)
- ⭐ Stern-System basierend auf Koffeinaufnahme
- 🎮 3 Minigames: Flappy Bird, Snake, Memory
- 💾 Daten-Persistierung in JSON
